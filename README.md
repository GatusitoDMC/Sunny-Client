# Dependencias
- Fabric API 1.21.10 ( https://modrinth.com/mod/fabric-api )
- Fabric Language Kotlin 1.21.10 ( https://modrinth.com/mod/fabric-language-kotlin )
- Fabric Loader 1.21.10 ( https://fabricmc.net/ )

# ¿Como Instalar?
- Descarga y ejecuta Fabric Loader y descarga la version para 1.21.10
- Copia y pega Fabric API y Fabric languae Kotlin en tu carpeta .minecraft/mods
- Inicia Fabric Loader en tu launcher de Minecraft

# Dependencies
